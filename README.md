<a href="https://marvelous-druid-2c0899.netlify.app/" target="_blank">Live Demo</a>

## How to run the Front-end?
```
npm install
```
```
npm run dev
```
## How to run the Back-end?
```
npm install
```
```
node or nodemon server.js
```
